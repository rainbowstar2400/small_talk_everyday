# 英語スピーキング課題配信サービス — 仕様メモ

## これは何？

生徒が家に帰ってから、自動で英語のスピーキング課題が配信されるサービス。
毎日10分以内で取り組めて、教師側の追加労働はゼロ。

想定ユーザーの端末は iPad。

## ざっくり仕組み

- 毎日、生徒に課題が自動配信される
- 生徒は音声で回答する（スピーキング）
- システムが文字起こし・文法訂正・レベル提案をしてくれる（LLM利用）
- 土日は軽めのTips＋簡単な課題だけ

## レベル設計

生徒が自分でその週のレベルを選ぶ。

| レベル | やること | イメージ |
|---|---|---|
| 1 | テーマに沿って形式通りに話す | Tagaki的 |
| 1.5 | イラストを穴埋めで説明する | 1週間でストーリーが進む |
| 2 | イラストを自分の言葉で説明する | 同上 |
| 2.5 | 形式に沿ってエピソードトークする | テーマ＋形式を提供 |
| 3 | 自分の言葉でエピソードトークする | 2.5と同テーマ |
| 3.5 | 意見論述のフォーマットを練習する | 知識・テクニックも提示 |
| 4 | Topicについて自分で作論する | 小論文イメージ、1週間かけて仕上げる |

### レベル4の1週間の流れ

1日目：英語で話してみる
2日目：日本語で説明してみる（英語力ではなく思考力の確認）
3日目：システムが作った英訳を音読する
4日目：メモを見ながら英語で説明する
5日目：メモなしで英語で説明する

### 最終目標（仮）

英語開講授業に主体的に参加・発言できるレベル
